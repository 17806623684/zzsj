3.邮箱注册
在git bash界面输入如下内容即可完成邮箱的注册：

$ git config --global user.name "user.name"

（说明：双引号中需要你的用户名，这个可以随便输入，比如“zhangsan”）

$ git config --global user.email "yourmail@youremail.com.cn"

（说明： 双引号中需要输入你的有效邮箱，比如“12131312@qq.com”）
